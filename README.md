# Notes
Lecture notes from The Iron Yard. I began working on the pre-work exercises (approx 4 weeks of pre-work lessons) in April 2017. The pre-work is not included in this repository. The Iron Yard Front End Engineering bootcamp consisted of 3 months of intensive immersion. Lectures and peer programming was conducted on campus in downtown Orlando, Florida. My cohort started in June 2017. We started with Front-End Fundamentals for the first 4 weeks of the program. Followed by 4 weeks of Back-End Fundamentals. Then, each student could select a language focus. My language focus was React JS. Due to Hurricane Irma we spent 5 weeks in React JS. I was granted permission to have access to the Java language focus curriculum so that I could work on it on my own. The Java notes are included in this repository.
